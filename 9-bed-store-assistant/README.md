# Sales Chatbot

A Gradio-based AI chatbot for a bed and mattress store.

## What It Does

- Chat interface for customers
- Encourages purchases with sales info
- King-size beds: 60% off
- Other beds/mattresses: 50% off
- Redirects non-furniture queries

## Requirements

- Python 3.8+
- Gradio
- OpenAI API key

## Usage

```bash
python app.py
