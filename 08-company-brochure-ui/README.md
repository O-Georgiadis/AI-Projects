# Company Brochure Generator (UI)

A web-based interface for generating AI-powered company brochures using Gradio.

## What It Does

- Enter a company name and URL
- AI scrapes the landing page
- Generates a professional company brochure
- Choose between GPT-4.1-mini or Ollama (Llama 3.2)
- Streaming responses

## Requirements

- Python 3.8+
- Gradio
- OpenAI API key (for GPT)
- Ollama running locally (optional)

## Usage

```bash
python app.py
