# Python to C++ Transpiler

Convert Python code to high-performance C++ using AI.

## What it does
- Detect system information (CPU, OS, etc.)
- Get optimal compiler flags for your machine
- Use AI to port Python to C++
- Compile with maximum optimizations
- Benchmark Python vs C++ performance

## Requirements
- Python 3.8+
- openai
- C++ compiler (detected automatically)

## Usage
```bash
python main.py
