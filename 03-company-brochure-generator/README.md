# Company Brochure Generator

An AI-powered tool that scrapes a company website and generates a professional brochure using GPT.

## What It Does

- Fetches all relevant links from a company website
- Filters for About, Careers, and Company pages
- Scrapes content from relevant pages
- Generates a markdown brochure using GPT

## Requirements

- Python 3.8+
- OpenAI API key
- BeautifulSoup (for scraping)

## Usage

```bash
python main.py
```
