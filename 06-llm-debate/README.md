# LLM Debate

An experiment comparing different LLM personalities by having two chatbots debate each other.

## What It Does

- GPT-4.1-mini: snarky, argumentative
- Ollama (Llama 3.2): polite, agreeable
- Runs a 5-turn conversation between both bots

## Requirements

- Python 3.8+
- OpenAI API key
- Ollama running locally

## Usage

```bash
python main.py
