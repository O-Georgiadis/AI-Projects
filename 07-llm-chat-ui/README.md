# LLM Chat UI

A web-based chat interface built with Gradio to interact with GPT and Ollama models.

## What It Does

- Web UI for chatting with GPT-4.1-mini
- Web UI for chatting with Ollama (Llama 3.2)
- Dropdown to switch between models
- Streaming responses

## Requirements

- Python 3.8+
- Gradio
- OpenAI API key
- Ollama running locally

## Usage

```bash
python app.py
